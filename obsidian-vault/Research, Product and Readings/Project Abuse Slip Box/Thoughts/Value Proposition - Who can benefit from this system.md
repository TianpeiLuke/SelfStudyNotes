## Data Analyst

Abuse Slip Box is graph-based note system that can be queried and visualized. Its path would help to formulate a consistent narrative for deep dive purpose. 


## ML Scientist

Abuse Slip Box connects the inputs and outputs of online evaluation system. Its path allows us to search for data with similar patterns. Also it is a useful system to compare and verify the label of training data. The summarized and compressed text data would reduce the input token length of pretrained language models, improving its latency and accuracy. It also helps to reduce the noise in input data. Abuse Slip Box fits the framework of RAG for in-context learning. It would help LLM to improve the relevancy of its answer. 


## Human Investigator

The summarized text for customer contact would reduce the time of reading for human investigator. The association links would add to the existing links to build clusters that reflect more context of concessions. Human investigator can also use the note system to read and understand previous investigation results and to find the associated SOPs. This would help in investigator performance review and training.
