## Concept Definition

>[!important]
>**Name**: Trace Class Operator

>[!important] Definition
>An operator $A \in \mathcal{L}(\mathcal{H})$ is called **trace class** if and only if 
>$$
> \begin{align*}
> \text{tr}\left(|A|\right) = \text{tr}\left(\sqrt{ A^{*}A }\right) < \infty.
> \end{align*}
>$$
