## Concept Definition

>**Name**: Shattering of Set by Set Family

![[Intersection of Set Family to Set#^d95ae8]]

- [[Intersection of Set Family to Set]]

>[!important] Definition
>A set $C$ is said to be **shattered** by a *family of sets* $\mathscr{F}$ if
>$$
>|\mathscr{F} \cap C| = 2^{|C|}.
>$$
